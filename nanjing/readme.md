# 南京大学 往年题目

## 2019

给定一个十进制正整数n(0 < n < 1000000000)，每个数位上数字均不为0。n的位数为m。
现在从m位中删除k位(0<k < m)，求生成的新整数最小为多少？
例如: n = 9128456, k = 2, 则生成的新整数最小为12456

用DP：`dp[i][j]`: 指倒数第j个之后删除了i个元素的最小值
算法：

+ 先铺垫上不删除的结果。
+ 然后假设删除，取j-1的值。比对不删除的结果，取最小值。

|     | 0       | 1      | 2     |
| --- | ------- | ------ | ----- |
| 0   | 0       | x      | x     |
| 1   | 6       | 0      | x     |
| 2   | 56      | 5      | 0     |
| 3   | 456     | 45     | 4     |
| 4   | 8456    | 456    | 45    |
| 5   | 28456   | 2456   | 245   |
| 6   | 128456  | 12456  | 1245  |
| 7   | 9128456 | 128456 | 12456 |
